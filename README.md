# five.js

A library to overcomplicate `5`.

### Usage
##### Require the module
```javascript
var five = require('five');
```

##### Basic 5
```javascript
five(); // 5
```

##### Addition
```javascript
five() + five(); // 10
```

##### Multiplication
```javascript
five() * five(); // 25
```

##### Division
```javascript
five() / five(); // 1
```

### Development
##### The code
All of the logic & heavy lifting is achieved in one self-contained file: 
```
./index.js
```

##### Tests
The tests are written with `mocha` and can be run with
```
mocha
```
or
```
npm test
```
